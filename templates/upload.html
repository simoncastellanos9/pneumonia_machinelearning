Methodology

This is a pneumonia detection tool and was developed using a convolutional neural (CNN/vgg16) network architecture used for deep learning. The tool recognizes x-ray patterns from an uploaded image file computing either a positive or negative result for pneumonia.

This CNN model is from the Keras library and was trained and fitted using publicly available x-ray images provided in Kaggle: 5,216 chest x-ray images (1,341 negative, 3,875 positive). In addition, the model was trained at various epoch stages while monitoring for the most optimal accuracy scores for both precision and recall. Kaggle resource for x-ray data images: https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia

Model Accuracy

As with any CNN model, the above model predictions are not always accurate. Model training accuracy is 92%, with less favorable testing accuracy scores of 62% precision and 67% recall.

During the model testing phase (624 test images), the most optimal CNN model was generated while using ~1-30 epochs cycles. Model accuracy declines significantly after 50 plus epoch cycles due to overfitting

Report

The report predicts x-ray images as either negative (0) or positive (1) for pneumonia. A trained hdf5 model file containing weights for each layer, is compared against an uploaded “test” x-ray file using the above screen. Once the report is submitted, a negative or positive prediction is displayed.
